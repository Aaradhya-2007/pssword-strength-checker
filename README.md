# Password Strength Checker (CLI)

A simple command-line tool written in Python that checks the strength of a password.

## Features
- Checks password length
- Detects:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Rates password strength:
  - Weak
  - Medium
  - Strong

## How to run
1. Make sure Python is installed.
2. Open terminal in the project folder.
3. Run:

python password_checker.py

## Example
Enter your password: Hello123!
Password is long
Uppercase: True
Lowercase: True
Digit: True
Special: True
Password strength: Strong
